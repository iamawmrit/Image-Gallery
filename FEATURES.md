# Features

Gallery is packed with features designed for speed, privacy, and a seamless user experience.

## Core

- **Masonry Grid Layout**: Automatically arranges photos in a responsive, visually pleasing grid that respects original aspect ratios.
- **Local & Private**: All processing happens on your machine. No cloud uploads, no tracking, no data collection.
- **Blazing Fast**: Optimized for performance with virtual scrolling and efficient thumbnail generation, handling thousands of photos smoothly.
- **Dark Mode**: A sleek, modern dark interface that's easy on the eyes.

## Organization

- **Sidebar Navigation**: Browse your file system with a native-style sidebar.
- **Folder Watching**: Automatically detects new photos added to watched folders.
- **Sorting**: Sort by Name, Date Modified, Date Created, Size, or Type.
- **Search**: Instantly filter photos by filename.
- **Multi-Select**: Select multiple photos for batch actions (export, delete).

## Viewer

- **Immersive View**: Full-screen image viewer with zoom and pan capabilities.
- **Filmstrip**: Quickly navigate between photos without leaving the viewer.
- **EXIF Data**: View detailed metadata (Camera, ISO, Aperture, Location, etc.) in a dedicated panel.
- **Slideshow**: Auto-play your photos with customizable intervals.

## Editor

A powerful built-in image editor to enhance your photos without leaving the app.

### Adjustments
- **Brightness**
- **Contrast**
- **Saturation**
- **Blur**
- **Grayscale / Sepia**
- **Invert**

### Filters
- **Presets**: Apply instant looks like Vintage, B&W, Warm, Cool, etc.

### Tools
- **Crop**: Freeform or aspect-ratio locked cropping.
- **Drawing**:
  - **Pencil**: Freehand drawing.
  - **Shapes**: Line, Arrow, Rectangle, Circle.
  - **Text**: Add text annotations with custom colors.
  - **Blur Tool**: Blur specific areas of the image.
- **Undo/Redo**: Full history support for all edits.

## Technical

- **Electron**: Cross-platform desktop runtime.
- **Sharp**: High-performance image processing (resizing, converting).
- **Fabric.js**: Advanced canvas manipulation for the editor.
- **Better SQLite3**: Fast, local database for metadata caching.
