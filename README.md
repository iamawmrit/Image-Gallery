Built because the default macOS gallery didn't meet my needs for speed and flexibility. Gallery is designed to be fast, beautiful, and completely local.

## Installation

### Download
Download the latest version from the [Releases](https://github.com/iamawmrit/Gallery/releases) page.

### Build from Source

Prerequisites: Node.js (v18+) and npm.

1. Clone the repository:
   ```bash
   git clone https://github.com/iamawmrit/Gallery.git
   cd Gallery
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application in development mode:
   ```bash
   npm start
   ```

4. Build for production:
   ```bash
   npm run dist
   ```
